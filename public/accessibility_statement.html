Accessibility statement

This accessibility statement applies to the Organisational registry (https://organisation.it.helsinki.fi) website.
This website is governed by the Act on the Provision of Digital Services to provide an accessible website according to WCAG 2.1 A and AA standards.
The statement has been updated on 28.08.2023.

The accessibility of this website has been audited by the University of Helsinki.

Accessibility status of the digital service

Partially meets the WCAG 2.1 A and AA accessibility requirements.

Non-accessible digital service content

The website is not yet fully compliant with the requirements

  • Perceivable: Insufficient or missing labels in the organisation hierarchies dropdown list

    The selected hierarchies in the organisation hierarchies dropdown list are missing labels, or they are not set correctly.

    WCAG accessibility criteria not met
    • 1.3.1 Info and Relationships
    • 2.5.3 Label in Name

  • Perceivable: Some content is not visible when page zoom is set to 400 %.

    Unit information is blocked by other elements when page zoom is set to 400 %.

    WCAG accessibility criteria not met
    • 1.4.10 Reflow

  • Operable: The on-focus visual contrast of the "show history" and "show coming" switches is insufficient.

    When the history and future switches receive focus, the visual contrast is not sufficient to indicate that focus has been received.

    WCAG accessibility criteria not met
    • 2.4.7 Focus Visible

  • Understandable: Screen readers do not announce the date format.

    The date fields do not indicate the proper date format for screen readers.

    WCAG accessibility criteria not met
    • 3.3.2 Labels or Instructions

Supervisory authority

If you notice any accessibility issues on the website, please provide feedback to us (the site administrator) first. We will reply within 14 days.
If you are not satisfied with our response or do not receive any response within two weeks, you can file a report with the Regional State Administrative Agency for Southern Finland. The page of the Regional State Administrative Agency for Southern Finland contains a detailed description of how the report can be filed and how the matter will be processed.
